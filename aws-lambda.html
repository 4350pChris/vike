AWS Lambda

From an architectural point of view, a vite-plugin-ssr app is simply a Node.js application.

This means we can use any AWS Lambda Node.js deploy tool, for example:

  • Claudia.js
  • Apex Up
  • Serverless Framework
  • SAM
In production, a vite-plugin-ssr app is simply two server middlewares: one middleware that serves the static files living at dist/client/, and a second middleware that server-side renders our pages.
