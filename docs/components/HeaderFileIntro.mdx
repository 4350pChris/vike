Vike transforms `.h.js` files from this:

```js
// +config.h.js

import Page from './some-file'
import onBeforeRender from './some-other-file'

export default {
  Page,
  onBeforeRender
}
```

Into that:

```js
const Page = 'import:./some-file:default';
const onBeforeRender = 'import:./some-other-file:default';

export default {
  Page,
  onBeforeRender
}
```

> This transformation is only for config files. (Any `.h.js` file is a `+config.h.js` file.)
