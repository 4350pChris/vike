Vike extensions provide automatic integration with tools.

## UI frameworks

<Link text="UI framework" href="/ui-framework" /> integrations.

 - [`vike-react`](https://github.com/vikejs/vike-react)
 - [`vike-vue`](https://github.com/vikejs/vike-vue)
 - [`vike-solid`](https://github.com/vikejs/vike-solid)
 - :construction: [`vike-svelte`](https://github.com/brandonxiang/vike-svelte)
 - :construction: [`vike-angular`](https://github.com/vikejs/vike-angular)

> You can use [Bati](https://batijs.github.io/) to easily scaffold a Vike app using one of these extensions.

## Others

Extensions for other kind of integrations are work-in-progress, see [#1263](https://github.com/vikejs/vike/issues/1263).
