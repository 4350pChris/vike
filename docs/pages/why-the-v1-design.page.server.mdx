The <Link text="V1 design" href="/migration/v1-design" /> enables <Link href="/vike-packages">Vike packages</Link> to extend Vike for deeply integrating tools.

```js
// /pages/admin/+config.h.js

import vue from 'vike-vue'
import graphql from 'vike-vue-apollo'

export default {
  // Render the Admin Panel as SPA
  ssr: false,
  // Use Vue + GraphQL for the Admin Panel
  extends: [vue, graphql]
}
```

You can even choose packages on a page-by-page basis:

```js
// /pages/product/@id/+config.h.js

import react from 'vike-react'
import telefunc from 'vike-react-telefunc'

export default {
  // Use React + Telefunc for the product pages
  extends: [react, telefunc]
}
```
