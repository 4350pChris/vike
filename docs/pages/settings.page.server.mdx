List of *built-in* Vike settings (you can <Link href="/meta">create your own settings</Link>).

## Basic

- [**`Page`**](/Page): <TextType>unknown</TextType> The root UI component of the page.
- [**`route`**](/routing): <TextType>string | Function</TextType> The page's route.
- [**`prerender`**](/prerender-config): Whether to <Link text="pre-render" href="/preredering" /> the page, and pre-rendering settings.
- [**`client`**](/client): <TextType>string</TextType> Additional client entry.
- [**`redirects`**](/redirects): Permanent redirections.
- [**`prefetchStaticAssets`**](/prefetchStaticAssets): Link prefetching settings.
- [**`filesystemRoutingRoot`**](/filesystemRoutingRoot): Change the URL root of Filesystem Routing (usually for <Link href="/file-structure#domain-driven">domain-driven file structure</Link>).

## Advanced

> Most users don't need to know about these configs. (They are usually only used by <Link href="/vike-packages" noBreadcrumb />.)

- [**`meta`**](/meta): Create new or modify existing configurations.
- [**`passToClient`**](/passToClient): <TextType>string[]</TextType> Determines what `pageContext` properties are sent to the client-side.
- [**`clientRouting`**](/clientRouting): <TextType>boolean</TextType> Enable Client Routing.
- [**`hydrationCanBeAborted`**](/hydrationCanBeAborted): <TextType>boolean</TextType>  Whether the <Link text="UI framework" href="/ui-framework" /> allows <Link href="/hydration">hydrations</Link> to be aborted.
