The `filesystemRoutingRoot` config option allows us to change the URL root of Filesystem Routing.

> **Don't use `filesystemRoutingRoot` to change the base URL**, see <Link href="/base-url" /> instead.

> `filesystemRoutingRoot` is usually used for <Link href="/file-structure#domain-driven">domain-driven file structure</Link>.

## Example

```js
// /marketing/+config.h.js

export default {
  // We route our marketing pages to `/*` instead of `/marketing/*`
  filesystemRoutingRoot: '/'
}
```

```
/marketing/+config.h.js
/marketing/pages/index/+Page.js
/marketing/pages/about/+Page.js
/marketing/pages/jobs/+Page.js
```

Result:

```
PROJECT FILES                               URL
/marketing/pages/index/+Page.js             /
/marketing/pages/about/+Page.js             /about
/marketing/pages/jobs/+Page.js              /jobs
```
