Environment: browser.

The `<ClientOnly>` wrapper enables you to load and render a component only on the client-side.

> It is currently available through [`vike-react`](https://github.com/vikejs/vike-react) and [`vike-solid`](https://github.com/vikejs/vike-solid), and [`vike-vue`](https://github.com/vikejs/vike-vue).

It is particularly useful for optimizing performance with heavy components, and for components that are meant for client-side only, such as those using browser-specific features. See <Link href="/client-only-components" />.
